      • 问题：解释单链表和双链表的区别，链表的插入和删除操作。
        • Question: Explain the difference between a singly linked list and a doubly linked list, and the insertion and deletion operations in a linked list.
      • 要点：
        • 单链表：每个节点指向下一个节点
          • Singly linked list: Each node points to the next node
        • 双链表：每个节点指向前一个和下一个节点
          • Doubly linked list: Each node points to both the previous and next node
        • 插入/删除操作：O(1)（需要找到插入/删除位置）
          • Insertion/Deletion: O(1) (once the insertion/deletion position is found)

    • 哈希表（Hash Table）
      • 问题：解释哈希表的工作原理以及如何处理碰撞（collision）。
        • Question: Explain how a hash table works and how collisions are handled.
      • 要点：
        • 哈希函数将键映射到唯一的物理地址
          • Hash function maps keys to unique physical addresses
        • 碰撞处理方法：链地址法（chaining）、开放地址法（open addressing）
          • Collision handling methods: Chaining, open addressing
    • 树（Tree）
      • 问题：解释二叉树和二叉搜索树的区别，并描述树的遍历方法。
        • Question: Explain the difference between a binary tree and a binary search tree, and describe tree traversal methods.
      • 要点：
        • 二叉树：每个节点最多有两个子节点
          • Binary tree: Each node has at most two children
        • 二叉搜索树：左子节点小于根节点，右子节点大于根节点
          • Binary search tree: Left child is less than the root, right child is greater than the root
        • 遍历方法：前序、中序、后序、层序
          • Traversal methods: Preorder, inorder, postorder, level order
    • 图（Graph）
      • 问题：解释图的基本概念和常用遍历算法。
        • Question: Explain the basic concepts of a graph and common traversal algorithms.
      • 要点：
        • 图由顶点和边组成
          • Graph consists of vertices and edges
        • 遍历算法：深度优先搜索（DFS）、广度优先搜索（BFS）
          • Traversal algorithms: Depth-first search (DFS), breadth-first search (BFS)
  • 回归分析（Regression Analysis）
    • 线性回归（Linear Regression）
      • 问题：解释线性回归的原理及其假设。
        • Question: Explain the principle of linear regression and its assumptions.
      • 要点：
        • 目标：最小化均方误差
          • Goal: Minimize mean squared error
        • 假设：线性关系、误差正态分布、独立同分布
          • Assumptions: Linear relationship, normally distributed errors, independent and identically distributed errors
    • 多项式回归（Polynomial Regression）
      • 问题：解释多项式回归与线性回归的区别。
        • Question: Explain the difference between polynomial regression and linear regression.
      • 要点：
        • 多项式回归拟合的是多项式曲线
          • Polynomial regression fits a polynomial curve
        • 处理非线性关系
          • Handles nonlinear relationships
    • 岭回归（Ridge Regression）
      • 问题：解释岭回归的正则化作用及其与线性回归的区别。
        • Question: Explain the regularization effect of ridge regression and its difference from linear regression.
      • 要点：
        • 加入L2正则化项
          • Adds L2 regularization term
        • 防止过拟合
          • Prevents overfitting
    • LASSO回归（LASSO Regression）
      • 问题：解释LASSO回归的稀疏性及其与岭回归的区别。
        • Question: Explain the sparsity of LASSO regression and its difference from ridge regression.
      • 要点：
        • 加入L1正则化项
          • Adds L1 regularization term
        • 产生稀疏模型（一些权重为零）
          • Produces sparse models (some weights are zero)
  • 逻辑回归（Logistic Regression）
    • 问题：解释逻辑回归用于分类的原理。
      • Question: Explain the principle of logistic regression for classification.
    • 要点：
      • 使用sigmoid函数
        • Uses sigmoid function
      • 处理二分类问题
        • Handles binary classification problems
  • 正则化（Regularization）
  • L1和L2正则化（L1 and L2 Regularization）
    • 问题：解释L1和L2正则化的作用及其区别。
      • Question: Explain the effects and differences between L1 and L2 regularization.
    • 要点：
      • L1正则化：引入稀疏性，产生稀疏模型
        • L1 regularization: Introduces sparsity, produces sparse models
      • L2正则化：防止权重过大，减小过拟合
        • L2 regularization: Prevents weights from getting too large, reduces overfitting
  • 过拟合和欠拟合（Overfitting and Underfitting）
    • 问题：解释过拟合和欠拟合及其解决方法。
      • Question: Explain overfitting and underfitting and their solutions.
    • 要点：
      • 过拟合：模型过于复杂，训练误差低，测试误差高
        • Overfitting: Model is too complex, low training error, high testing error
      • 欠拟合：模型过于简单，训练误差高
        • Underfitting: Model is too simple, high training error
      • 解决方法：正则化、增加数据、交叉验证
        • Solutions: Regularization, adding data, cross-validation
  • 数据增强（Data Augmentation）
    • 问题：解释数据增强的作用及其常用方法。
      • Question: Explain the purpose of data augmentation and common methods.
    • 要点：
      • 通过随机裁剪、旋转、添加噪声等方法生成更多数据
        • Generate more data by random cropping, rotation, adding noise, etc.
      • 增强模型的泛化能力
        • Enhance model generalization
  • 交叉验证（Cross-validation）
    • 问题：解释K折交叉验证的原理及其作用。
      • Question: Explain the principle and purpose of K-fold cross-validation.
    • 要点：
      • 将数据分为k组，交替训练和测试
        • Divide data into k groups, alternately train and test
      • 提高模型的泛化能力
        • Improve model generalization
  • Dropout
    • 问题：解释Dropout在神经网络中的作用及其原理。
      • Question: Explain the role and principle of Dropout in neural networks.
    • 要点：
      • 随机丢弃网络连接
        • Randomly drop connections in the network
      • 防止过拟合，增强模型鲁棒性
        • Prevent overfitting, enhance model robustness
